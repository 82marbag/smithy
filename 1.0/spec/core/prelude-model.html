4. Prelude model¬∂

All Smithy models automatically include a prelude. The prelude defines various simple shapes and every trait defined in the core specification. Shapes defined in the prelude can be referenced from within any namespace using a relative shape ID.

Tip

Prelude shapes SHOULD be used when possible to minimize duplication in models.
